# YD Interactive website

A free static website prepared for GitHub Pages.

## Publish with a free GitHub organization

1. Sign into GitHub with your existing account.
2. Create a free organization named `ydinteractive`, if that name is available:
   - Profile picture → Settings → Organizations → New organization
   - Select GitHub Free
   - Contact email: `ydinteractive.games@gmail.com`
3. Under the organization, create a **public** repository named exactly:
   `ydinteractive.github.io`
4. Upload every file from this folder to the root of that repository.
5. Open the repository:
   Settings → Pages
6. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
7. Save. The site should appear at:
   `https://ydinteractive.github.io`

If the organization name is unavailable, use the existing GitHub account and a
repository named `ydogan22.github.io`. The website address will then be
`https://ydogan22.github.io`.

## Before store submission

The privacy policy must match the actual game build.

Update `privacy.html` before enabling any of these:
- Unity Analytics
- Unity Ads / LevelPlay / AdMob or another advertising SDK
- Crash reporting
- Cloud saves or player accounts
- Game Center / Google Play Games Services
- In-app purchases
- Any third-party analytics or attribution SDK

Also add the privacy-policy link inside the game's Settings screen.
